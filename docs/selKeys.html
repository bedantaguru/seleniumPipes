Selenium key mappings

Usage

selKeys

Format

A named list. The names are the descriptions of the keys. The values are the "UTF-8" character representations.

Source

http://code.google.com/p/selenium/wiki/JsonWireProtocol#/session/:sessionId/element/:id/value

Description

This data set contains a list of selenium key mappings. The key mappings are outlined at https://github.com/SeleniumHQ/selenium/wiki/JsonWireProtocol#sessionsessionidelementidvalue. selKeys is used when a sendKeys variable is needed. sendKeys is defined as a list. If an entry is needed from selKeys it is denoted by key.
